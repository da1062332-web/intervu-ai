# Observability Report

## Metrics Captured

- `request_count`: Total requests processed.
- `request_duration`: Used to calculate average duration of API calls.
- `error_count`: Tracks exceptions.
- `active_requests`: Concurrency tracker.
- `http_status_distribution`: Provides a histogram of status codes (e.g., 200, 400, 429, 500).

## Implementation Details

Metrics are collected via a vendor-neutral `ObservabilityInterceptor`. No heavy third-party agent is tightly coupled, keeping the service lightweight while still providing `/health/metrics` for scrapers.

## Logging

Structured JSON logging is enabled using `@intervu-ai/shared-logger` (Pino). The `requestId` generated by `CorrelationMiddleware` successfully propagates across the request lifecycle and appears in every log line. Sensitive fields are successfully masked.
